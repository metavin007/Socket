รวมคำสั่งที่ใช้ทำงาน

cordova create ชื่อโฟเดอร์App ชื่อแพคเกจ ชื่อApp

cordova platform add android / cordova platform add ios

cordova platform remove ios

cordova build

cordova build android เฉพาะ platform

cordova emulate android คำสั่งนี้ จะ build และ run กับอุปกรณ์จำลองโทรศัพท์ในเครื่องPC

cordova run android / cordova run ios ทดสอบrunกับอุปกรณ์จริง

cordova serve เปิดเซอร์วิสเป็นเซิฟเวอร์โดยเข้า localhost:8000

cordova prepare เป็นการ copy HTML CSS Javascript ไปใส่แต่ละ platform
